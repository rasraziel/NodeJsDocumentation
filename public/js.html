Variables

JavaScript variables are containers for storing data values. Before 2015, using the var keyword was the only way to declare a JavaScript variable. The 2015 version of JavaScript (ES6) allows the use of the const keyword to define a variable that cannot be reassigned, and the let keyword to define a variable with restricted scope(block scope). JavaScript variables can hold numbers like 100, strings like "John Doe and many more types of data, like undefined, null or boolean. Strings are written inside double or single quotes. Numbers are written without quotes. If you put a number in quotes, it will be treated as a text string. Below is an example of variable declaration:

let pi = 3.14;
let person = "John Doe";
let answer = 'Yes I am!';
const firstName = "Walter";
const lastName = "White";
const year = "2020";
const number = 1;

We can use these variable for different operations. With string literals we can format a string exactly how we want it. For example this line of code:

console.log(`My first name is ${firstName} and my last name is ${lastName}`);

Will log My first name is Walter and my last name is White on the console. And the following line of code:

console.log(+ year + number);

will first cast the string year into a number, make the addition and log the result as 2021.

Strings

With strings we can perform many kinds of operations. For example these lines of code: 

const anotherNumberOne = "1.10";
const anotherNumberTwo = "2.30";
console.log((+numberOne + +numberTwo).toFixed(2));

will first cast the strings anotherNumberOne and anotherNumberTwo into numbers, make the addition and log the result with 2 decimal points as 3.40.

const letters = "abc";
console.log(letters.indexOf('c'));

The above lines of code will log the index of the letter 'c' in the string, giving us 2.

console.log(fact.replace('b','B'));

And this one would substitute the lowercase 'b' with an uppercase 'B' logging aBc on the console.

Objects

JavaScript objects are containers for named values called properties or methods. The values are written as key:value pairs (key and value separated by a colon). We can define (and create) a JavaScript object with an object literal { }: 

let person = {
  firstName: "John",
  lastName: "Doe",
  age: 50,
  eyeColor: "blue"
};

We can access the onject properties in 2 ways, with the . operator or using square brackets [ ]:

person.lastName;
person["lastName"];

As with other programming languages, objects can also contain functions:

let person = {
 firstName: "John",
 lastName : "Doe",
 id       : 5566,
 fullName : function() {
   return this.firstName + " " + this.lastName;
 }
};

We can also add or remove properties at will:

const thisSong = {"description": "The best song in the world."};
delete thisSong.description;
thisSong.about = "Just a tribute.";
console.log(thisSong);

The above lines of code will remove the "description" property and add a new "about" property to the thisSong object.

Arrays and loops

JavaScript arrays are used to store multiple values in a single variable and we can access the values by referring to an index number. Using an array literal is the easiest way to create a JavaScript Array: 

let cars = ["Saab", "Volvo", "BMW"];
let name = cars[1];
console.log(name);

The above code creates an array with car names, then retrieves the item at index 1 and logs it on the console.

We can change a value in an array:

cars[0] = "Opel";

The real strength of JavaScript arrays are the built-in array properties and methods:

let length = cars.length;   // The length property returns the number of elements
let sorted = cars.sort();   // The sort() method sorts arrays

The difference between arrays and objects is that arrays use numbered indexes, while objects use named indexes.

Loops are handy, if you want to run the same code over and over again, each time with a different value. Often this is the case when working with arrays:

for (i = 0; i < cars.length; i++) {
  console.log(cars[i]);
}

The above code will log all the cars in the array on the console.
